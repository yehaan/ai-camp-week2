# plan.md — 2주차 자동화 MVP 컨텍스트

> 1주차 나침반: 일본 사용자 대상 UX 라이팅 및 번역 보조 AI

## 이번 주 만들 산출물 1개

한국어 **서비스 UX 문구 전반**을 일본어로 현지화한 결과물 1세트.

1주차 반복 업무 중 **앱 서비스 문구 현지화**를 다룬다. 버튼·라벨 같은 마이크로카피에 그치지 않고
안내문·약관·동의 문구·FAQ·푸시 알림·에러 문구까지 — 서비스에서 번역이 필요한 문구 전반이 대상이다.
도메인은 일본 사용자 대상 **뷰티·의료·시술 예약 앱**.
골든 데이터는 학생 본인이 실제로 진행한 한국어→일본어 현지화 프로젝트의 산출물 100건을 익명화하고,
**문구 유형 8개 카테고리**(안내·설명 / 마이크로카피·라벨 / 상태·피드백 / 약관·동의 / 취소·환불 정책 / 알림·푸시 / FAQ·Q&A / 에러·예외)로 분류해 사용한다.
이유: 실제 작업물이라 신뢰도가 가장 높고, 유형별로 분류돼 있어 번역 보조 AI가 유형별 규칙을 학습하기 좋고 격차 분석도 정확하다.

## AI / 사람 역할 경계 (한 줄)

**AI는 일본어 UX 마이크로카피 초안과 대안 표현을 생성하고, 사람은 브랜드 톤·업계 관용어·법률 표현·최종 릴리스 여부를 판단한다.**

| AI가 맡는 일 | 사람이 최종 확인하는 일 |
|---|---|
| 한국어 원문 → 일본어 초안 생성 | 브랜드 톤·서비스 페르소나 적합성 |
| 자연스러운 대안 표현 2~3개 추천 | 뷰티·의료 업계 관용어의 적절성 |
| 글자수 제한·경어 레벨·용어 일관성 체크 | 법률·약관·필수/선택 라벨의 정확성 |
| 표기 흔들림(敬体/常体)·placeholder 보존 점검 | 실제 화면 맥락의 사용자 경험·릴리스 결정 |

## 과제 범위에서 제외 (안전 규칙)

- 실제 앱·서비스에 문구를 반영하지 않음
- 발송·결제·로그인·외부 시스템 조작 없음
- 골든 데이터는 **익명화**: 브랜드명 `서비스`/`サービス`, 고객사명 `○○○`로 치환
- 미번역 행·내부 메모·타사명 유출 행·환자/고객 데이터·신체부위 나열 목록은 제외
- 보조 레퍼런스로 공개 서비스([TRIBEAU](https://tribeau.jp))의 공개 UI 문구도 참고했으나, 후기·가격·실명은 제외
