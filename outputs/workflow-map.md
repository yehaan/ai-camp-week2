# outputs/workflow-map.md — 업무 순서도 (선택 산출물)

> 일본어 UX 문구 현지화 반복 업무의 흐름 (버튼·라벨부터 안내문·약관·FAQ·푸시까지).
> AI 위임 구간과 사람 판단 구간을 표시한다.

```
[1] 한국어 원문 + 화면 컨텍스트 수집        ← 사람 (입력 준비)
        │
        ▼
[2] 글자수 제한·톤 가이드 정리              ← 사람 (컨텍스트 작성)
        │
        ▼
[3] 일본어 초안 + 대안 표현 2~3개 생성       ← AI ★위임
        │
        ▼
[4] 글자수·경어 레벨·품사 규칙 자동 체크      ← AI ★위임
        │
        ▼
[5] 브랜드 톤·법률/결제 표현 검수            ← 사람 ★판단
        │
        ▼
[6] 최종 문구 확정 + 릴리스 반영 여부 결정    ← 사람 ★판단
```

## 구간 요약

- **AI 위임 구간**: [3] 초안·대안 생성, [4] 규칙 체크
- **사람 판단 구간**: [1][2] 입력 준비, [5] 톤·법률 검수, [6] 최종 결정
- 3주차 제약 조건은 [3][4] 구간의 AI 출력 품질을 끌어올리는 데 쓰인다.
